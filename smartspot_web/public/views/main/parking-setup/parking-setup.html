Setup Instructions

Requirements

  • RaspberryPi 2
  • {{parking.spots.length}} HC-SR04 ultrasonic sensors
  • {{parking.spots.length}} x 1 multiplexer (or a multiplexer with more channels)
  • one 1 kΩ (or more) resistor

Wiring

The following sketch shows the wiring model for a parking with 1024 ultrasonic sensors. For parkings with less sensors, start connecting the multiplexer select pins to the RaspberryPi's GPIO02 onwards.

Download the Driver

Main Driver Test Driver

* Main Driver: This is the main driver that reads all sensors of the parking and sends the data over to the central server.

* Test Driver: This is a driver which can be used for testing purposes. This driver does not read sensors, but only sends fake and random data over to the server.

Run the Driver

  • Transfer the driver into the RaspberryPi
  • Make it executable: $ sudo chmod +x driver.py
  • Run the driver: $ ./driver.py
